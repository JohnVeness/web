Good news!  

We are excited to announce the release of OsmAnd version 4.8 update for Android devices, which is already available for download on Google Play. This latest update introduces many new features and improvements to enhance your navigational experience.

**What's new**

- Introduced Main Navigation widget,
- Added a full-screen [gallery viewer](#images-gallery) for Wikimedia images,
- Introduced a new [plugin "Vehicle Metrics"](#vehicle-metrics) to monitor vehicle performance using the OBD-II protocol,

- Added new route layer ["Dirt Bike trails"](https://github.com/osmandapp/OsmAnd/issues/18756),
- Added information about surrounding areas for the selected point,
- Separated visibility parameters for Terrain colorization options,
- Added a tappable icon to display access or other road or path attributes,
- Fixed "Auto-record track during navigation",
- Fixed issue with fliped coordinates in RTL,

## Images Gallery

Added on [Map Contex menu](https://osmand.net/docs/user/map/map-context-menu#online-photos) full-screen gallery viewer for Wikimedia images.

![Gallery](./img/gallery.png)

## Vehicle Metrics

The new Plugin - Vehricle Metrics was added to the [Plugins menu](https://osmand.net/docs/user/plugins/). It monitors key vehicle metrics using the [OBD-II protocol](https://en.wikipedia.org/wiki/OBD-II_PIDs). New widgets for this data were added too.

![OBD](./img/obd.png) ![OBD](./img/obd_1.png)

## GPX Track activities

The new feature of adding **Activity** to a recorded GPX track improves navigation and track management (filter). This increases the convenience of working with track recordings.

![3D track](./img/track_activity.png)

## Customizable Map buttons

No you can modify [the appearance of Quick Action buttons](https://osmand.net/docs/user/widgets/quick-action#quick-action-button-appearance) and Default buttons through the three-dot menu: Menu ‚Üí Configure screen ‚Üí Custom buttons or Default buttons ‚Üí choose needed buttons ‚Üí three-dot menu ‚Üí Appearance (chagne icon, corner radius, size, background opacity).

Added [grid for buttons](https://github.com/osmandapp/OsmAnd/issues/19164).

![Button appearance](./img/button_appearance.png) ![Button appearance](./img/button_appearance_1.png)


## Widget Context Menu

The [**Widget Context menu**](https://osmand.net/docs/user/widgets/configure-screen.md#panel-widgets-settings) allows you to manage and configure widgets directly from the map screen, providing *a quick way to organize panels*. You can access this menu by ***long tapping*** on any widget to customize the layout of widgets without leaving the map.

![Widget context menu](@site/static/img/widgets/widget_context_menu_andr.png)

## New actions for QA

Added [new actions](https://osmand.net/docs/user/widgets/quick-action#action-types): 
1. Action for Lock/Unlock screen: _Menu ‚Üí Configure screen ‚Üí Custom buttons ‚Üí + ‚Üí Add acton ‚Üí Inteface ‚Üí Lock screen_
2. Actions for Trip recording: _Menu ‚Üí Configure screen ‚Üí Custom buttons ‚Üí + ‚Üí Add acton ‚Üí My Places ‚Üí Start/Pause - Trip recording, Start new trip segment, Finish Trip recording, Save Recorded Trip & Continue_
3. Action for Show/Hide OSM edits: _Menu ‚Üí Configure screen ‚Üí Custom buttons ‚Üí + ‚Üí Add acton ‚Üí My Places ‚Üí Show/Hide OSM edits_

![Quick actions](./img/quick_action.png)

## Dirt Bike Trails

Added new Route options  - [Dirt bike trails](https://osmand.net/docs/user/map/vector-maps#routes): _Menu ‚Üí Configure map ‚Üí Routes ‚Üí Dirt bike routes_ 

Upon activation, display a colored route overlay for all paths marked with the dirtbike:scale tag, similar to the approach used for mtb:scale (Mountain bike).

![Dirt Bike legend](./img/dirt_bike.png)
